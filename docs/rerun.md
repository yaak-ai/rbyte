## Example visualization in [rerun](rerun.io)

```
# In a terminal window
just rerun
# open the following address in your browser

http://localhost:9090?url=ws://localhost:9877

just visualize dataset=yaak data_dir=/nas/drives/yaak/data logger=rerun_yaak
```
