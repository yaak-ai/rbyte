# rbyte

ryte is built for developers training end-to-end spatial intelligence (robotics) models.

## Installation

```bash
uv add git+https://github.com/yaak-ai/rbyte [--extra visualize]
```

## Usage

See [examples/config_templates/dataset](examples/config_templates/dataset).

## Development

### Setup

Requirements:
- [`uv`](https://github.com/astral-sh/uv)
- [`just`](https://github.com/casey/just)

```bash
just setup
```
